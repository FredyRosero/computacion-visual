## Python projects

### Virtual environment

Create a virtual environment
```bash
python -m venv .venv
source .venv/bin/activate
python
```

Check the python executable path
```python
import sys
print(sys.executable)
```

Install requirements
```bash
pip install -r requirements.txt
```

Deactivate the virtual environment
```bash
deactivate
```
